OATFD v1.0 Thesis Edition - short-path package

Extract this ZIP directly to a short folder, for example:
  C:\OATFD_V1\

Recommended run file:
  RUN_FILEPICKER_LFP_APP.bat

Do not extract under Downloads or folders with long nested names.

Important behavior:
- Target Path Keyword may be left blank for full-scope/all-file detection.
- Folder names such as Normal/Aktifitas Normal are not used as detection evidence.
- Clean unselected stale CSV is OFF by default and should be enabled only when intentionally removing stale parsed CSV from a previous case.
